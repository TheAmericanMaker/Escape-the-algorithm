# Screenshots

This folder holds the screenshot images referenced in the README's "Never used a terminal?" section.

## Needed screenshots

Capture these and save them at the filenames listed. Aim for ~800px wide, PNG format.

| Filename | What to capture |
|---|---|
| `windows-search-terminal.png` | Windows Start menu with "terminal" typed in the search box, showing "Terminal" in results |
| `windows-terminal-open.png` | A fresh Windows Terminal window (the black/purple window that opens) |
| `mac-spotlight-terminal.png` | macOS Spotlight search (⌘ Space) with "terminal" typed, showing Terminal.app highlighted |
| `mac-terminal-open.png` | A fresh macOS Terminal window |
| `linux-terminal-open.png` | A fresh terminal window on a common Linux distro (Ubuntu GNOME, etc.) |
| `windows-navigate-downloads.png` | Terminal showing `cd Downloads` then `dir` listing a CSV file |
| `mac-navigate-downloads.png` | Terminal showing `cd ~/Downloads` then `ls` listing a CSV file |
| `eta-running.png` | Terminal showing a successful `eta convert` run with the output box |
| `rss-reader-import.png` | An RSS reader's OPML import dialog with a file selected |

## Tips

- Use a clean desktop/terminal with no personal info visible
- Annotate with red arrows or circles to highlight the key UI element (optional but helpful)
- Compress with [Squoosh](https://squoosh.app/) or `optipng` before committing
